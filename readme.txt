Provides a method to include formatted conversations into posts.

== Description ==

Converation Viewer enables you to insert formatted converations into posts by linking to an
external text file. Using CSS, the converation is formatted nicely, keeping you from having to
spend time putting things into a ghastly table or some crazy markup.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
